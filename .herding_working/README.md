# Herding MCP 工作区

欢迎使用 Herding MCP（牧羊犬 MCP）！这里是您的智能任务管理工作区。

## 📁 目录结构

- `templates/` - 任务模板和计划模板
- `logs/` - 操作日志和更新记录
- `sessions/` - 任务会话和编辑历史
- `backups/` - 自动备份文件

## 🚀 快速开始

1. 使用 `get-project-info` 获取项目信息
2. 使用 `plan` 创建任务计划
3. 使用 `complete-task` 执行任务
4. 使用 `update-project-info` 更新状态
